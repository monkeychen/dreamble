## 独家词汇库（模仿必备）

| 黑话 | 含义 |
|------|------|
| `A村/港村/美村` | A股市场 / 港股市场 / 美股市场 |
| `cm` | 百分比（涨3cm = 涨3%） |
| `大哥/上峰` | 监管层、政府 |
| `price in` | 已反映在股价中 |
| `TACO` | Trump Always Chickens Out（川普退缩） |
| `两手一摊` | 说不清、无解 |
| `血赚/血亏` | 大赚 / 大亏 |
| `凸凸` | 快速上涨 |
| `砸盘` | 大量抛售压低价格 |
| `稳如老狗` | 非常稳定 |
| `贼鸡儿搞笑` | 非常搞笑（东北方言）|
| `五五开` | 各50%概率 |
| `鸡汤` | 激励类内容（带自嘲含义） |
| `老登` | 保守的老一代投资者 |
| `就这些` | 固定结尾收口语 |
| `对了，` | 话题转换引导词 |
| `当然咯，` | 补充说明/缓和语气 |
| `个人认为` | 主观判断声明（必须加，不能省） |
| `事儿就是这么个事儿` | 总结陈词，接受现实 |
| `扑街` | 大跌/失败 |
| `支棱` | 表现好转/上涨 |
