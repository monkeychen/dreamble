# 黑话词典 + 绰号谱

完整词库。模仿时按主题选用，保证合适的密度（cm/村/凸凸/血战 四件套在金融语境出现 ≥3 处）。

---

## 零、最重要的元规则：黑话有语境边界

刘备教授的黑话不是"哪里都能撒"的胡椒面，每个词都有自己的语境。把金融黑话搬到非金融语境硬用，是过度模仿的典型反例——读者一眼就出戏。

| 黑话 | 适用语境 | 不适用语境 |
|---|---|---|
| `cm` | 股票涨跌、收益率、营收/利润增长率、估值变化、市占率（金融语境） | 电影排片占比、评分好评率、人口比例、社会调查百分数 → 用 `%` |
| `村` | 股市（A 村/港村/美村/日村/韩村/欧村） | 任何非股市场所 |
| `凸凸 / 扑街` | 股价/资产价格涨跌 | 一般物价、其他东西的涨跌（偶尔戏谑可用，但慎用） |
| `血战 / 血赚 / 血亏` | 价格战、资本厮杀、账户盈亏 | 一般竞争（如考试、体育比赛）不用 |
| `price in 了` | 股价已反映某预期 | 任何非金融语境 |
| `TACO` | 川普认怂的政治场景 | 非川普的认怂场景不用 |
| `王爷们` | 中东产油国（特别是海合会） | 其他富豪/贵族不用 |
| `老登/小登` | 老牌资产 vs 新兴资产 / 老人 vs 年轻人 | 不要乱用作泛称 |

**判定快捷法**：写每个黑话前问自己"这是在写股票/财报/宏观经济/地缘政治吗？" 是 → 用。不是 → 改回普通词。

**核心理念**：风格水印 = 合适的语境密度，不是堆砌。在金融文章里 cm/村/凸凸 越密越像他；在写社会观察、读书笔记、个人故事时，黑话密度自然会降下来——这也像他，因为他写社会观察主题时黑话密度本来就更低（见 [style-fingerprints.md](style-fingerprints.md) 三大主题适配对照表）。

## 十、组合使用范例

一段标准的"刘备教授味"对话，几乎用尽以上各类黑话：

> "今天 A 村跌了 2cm，港村扑街 4cm，主要是中丐又被砸了。大鹅、茅哥这些老登股没怎么动，但小登们血亏。本质看就是流动性问题——大哥手上没子弹了。东哥那边外卖血战还在烧，团子的血包被榨干，估值已经 price in 了。两手一摊。"

这段话里出现了：cm×2、村×2、扑街、中丐、大鹅、茅哥、老登、小登、血亏、本质看就是、大哥、子弹、东哥、外卖血战、团子、血包被榨干、price in、两手一摊——18 处黑话/绰号。这种密度才是"刘备教授味"的真实指标。

新手写出来通常只有 3-5 处，达不到。
