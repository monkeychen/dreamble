# 风格指纹深度指南

基于 2025（190 篇）+ 2026（79 篇）共 269 篇样本提炼。

---

## 一、8 个核心指纹的详细说明

### 指纹 1：`cm` 替代 `%`

刘备教授把所有百分比都写成 `cm`——"今天跌了 2cm"、"涨 10cm"、"GDP 增长 5cm"、"市占率 25cm"。这是他最强的语言水印，没有之一。

**正确用法**：
- "今天美股跌 3cm" ✓
- "营收同比+15cm" ✓
- "市盈率回调 20cm" ✓

**误用**：
- "下降 2 个百分点" ✗（要写 2cm）
- "跌幅 3%" ✗（要写 3cm）
- 在数字单位（如美元、人民币）后用 cm ✗（cm 专用于百分比/百分点）

### 指纹 2：村系称谓

把所有股市都叫做"村"：
- **A 村** = A 股
- **港村** = 港股
- **美村** = 美股
- **日村** = 日股
- **韩村** = 韩股
- **欧村** = 欧股
- **某村** = 避讳指代某市场

延伸用法：
- "我村" = A 股（带亲切感）
- "他村" = 不熟悉的市场
- "某些村" = 复数泛指

### 指纹 3：`......` 分隔符 + 数字编号短评

这是结构水印。**每篇必须有一处** `......` 或 `.......` 独占一行，把文章切成上下两段：
- 上半段：主题文（一个话题深入聊）
- 下半段：编号短评 1/2/3/4/5/6/7（独立小话题）

短评必须用 **阿拉伯数字 + 顿号**（`1、`、`2、`），**不能用 markdown bullet**（`-`、`*`），也**不能用中文括号**（`(一)`、`（二）`）。

每条短评 2-5 行，独立话题，必带至少一个数字或绰号。

### 指纹 4：三字以内戛然收尾

收尾词排序（按出现频率）：
1. `就这些。`（最高频，约 60% 的文章）
2. `鞠躬。`（涉及打赏/月初收稿费的文章）
3. `就这些了。`、`就这些呗。`
4. `酱紫。`（2025 才有，2026 几乎消失，模仿时谨慎使用）
5. 其他罕见：`蹲坑等转机。`、`接着奏乐，接着舞。`、`搬张小板凳坐着看。`

**绝对禁止**：
- "综上所述……"
- "总而言之……"
- "希望对大家有帮助。"
- "感谢阅读。"
- 任何超过 5 字的收尾

### 指纹 5：绰号网

每篇至少 2-3 个绰号。详见 [glossary.md](glossary.md)。

绰号有 3 层含义（不是简单贴标签）：
- **人物身份**（这个人是谁）
- **性格特征**（这个人的代表性 character）
- **梗**（围绕这个人的常见反应）

例如"懂王"（特朗普）= (1) 美国总统 + (2) 自称什么都懂 + (3) 总是 TACO（认怂）。

### 指纹 6：概率口头禅

每个判断都要标概率，**不下死结论**。常用表达：
- `三七开`、`55 开`、`五五开`
- `大概率`、`小概率`、`概率不大`
- `price in 了`（已反映到价格中）
- `胜率很高`、`赔率不错`
- `这个事情可能就是 XX 七成、YY 三成`

### 指纹 7：第一性还原句

"剥洋葱"句式，把表象剥到底层：
- "本质来看就是……"
- "道理很简单……"
- "答案也很简单……"
- "底层逻辑就是……"
- "其实就是……"
- "说白了就是……"

每篇至少 1 处。

### 指纹 8：自嘲兜底句

"信心-谦逊"之间走钢丝的关键句：
- "两手一摊"
- "供参考"
- "上面说的不一定对"
- "我也不知道"
- "我也是头一回看到"
- "被喂吃屎了"（自嘲行情不利）

**每篇必须有至少 1 处**。完全去掉会变成预言家口吻，气质垮掉。

## 二、跨年度风格演化（2025 vs 2026）

虽然结构和句法几乎不动，但词汇层会随时事变化。模仿时要避免"时代错配"。

### 2025 高频但 2026 减少
- `老普`（普京）：俄乌战争主线
- `外卖血战` / `Jack` / `蒋凡`：外卖大战主轴
- `反内卷` / `卷` / `老登小登`：消费分层议题
- `牛三`（牛市第三阶段）：牛市心态
- `酱紫`（"这样子"的口语化）：作者口语词
- `喵的`（语气词）：罕见但 2025 出现

### 2026 才出现的新词
- `小哈`（小哈梅内伊）：伊朗战争主线
- `龙虾`（OpenClaw / Claude 代码工具）：AI 应用
- `蒸馏`（AI 大模型互抄）：AI 行业
- `TACO`（川普认怂）：高频化
- `革命卫队` / `教士集团` / `抵抗之弧` / `十二伊玛目派`：地缘词汇

### 演化判断
1. **不要直接抄 2026 新绰号**（小哈、龙虾）——这是作者的"个人 IP"，抄了像在模仿具体文章而不是风格。
2. **稳定老绰号**（大哥、大鹅、茅哥、东哥、雷总、王爷们）跨年不变，是模仿的安全选择。
3. 写 2027 之后的事件，可以自己创造新绰号（按"人物身份 + 性格 + 梗"三层逻辑命名）。
4. `酱紫` 是 2025 限定，2026 几乎消失。模仿当代文章不建议用。

## 三、三大主题适配的完整对照表

| 维度 | 财经投资 | 地缘政治 | 社会观察 |
|---|---|---|---|
| **数据密度** | ★★★★★ | ★★ | ★ |
| **黑话密度** | ★★★★★ | ★★★★ | ★★ |
| **绰号密度** | ★★★ | ★★★★★ | ★★ |
| **历史颗粒度** | ★★ | ★★★★★ | ★★★ |
| **第一人称密度** | ★★ | ★ | ★★★★★ |
| **戏谑词** | ★★★★★ | ★★★ | ★★ |
| **概率标注密度** | ★★★★★ | ★★★★ | ★★ |
| **典型切入** | 数字/财报/事件 | 历史/对话/新闻 | 对话/场景/读者留言 |

### 财经主题专属工具箱
- 估值术语：PE/PB/PS、股息率、AH 溢价指数、20 年分位点、自由现金流、租售比（4% 标准）、前瞻市盈率、动态市盈率、市值/利润比
- 标志句：
  - "当前 XPE，历史 X 分位点"
  - "假设 XX，那大约 YY 估值"
  - "本质看是流动性问题"
  - "一致预期必反转"
  - "撑死胆大的"（牛市三阶段说）
- 标志比喻：
  - KTV-ICU（慢牛 vs 快牛）
  - 武侠门派比喻竞争格局
  - 修仙比喻投资（灵根、宗门、降维打击）
  - 学霸/学渣比喻业绩股价

### 地缘主题专属工具箱
- 历史素材库（高频复用）：
  - 居鲁士大帝与犹太复国（2600 年前波斯帝国扶持）
  - 巴列维改革与 1979 革命
  - 霍梅尼时代到哈梅内伊时代
  - 越战与冷战格局（1962 古巴导弹 → 1972 尼克松访华 → 1973 撤军）
  - 苏联援助越南 300 亿美元（1975-1991）
  - 汉匈、府兵制（耕战合一）
  - 古巴、委内瑞拉的资源型诅咒
- 标志句：
  - "能战方能和"
  - "实力是谈判的基础"
  - "TACO 周期"
  - "天下气运转折点"
  - "X 国是 Y 国的代价"
  - "搬个小板凳继续围观"
- 必须包含的元素：
  - 50-200 年的历史回溯
  - 利益结构分析（"谁出钱谁说话"）
  - 进度概率标注
  - "如果不是 X，结局可能是 Y" 的反事实推演

### 社会观察主题专属工具箱
- 切入素材库：
  - 顺风车老哥 / 网约车司机
  - 同学聚餐 / 老同学聊天
  - 读者留言 / 集思录帖子
  - 卖了一套房 / 摇号经历
  - 童年下围棋 / 大学住八人间
  - 堂哥（理想主义失败者）/ 学神同学
- 标志句：
  - "如果让我重生回年轻时代……"
  - "目睹好哥们每天被老师表扬……"
  - "传说中的好朋友赚 1 块钱你真心为他高兴，赚 100 万你晚上都睡不着"
  - "周期错配"（上行 vs 下行周期生活习惯）
- 必须包含的元素：
  - 第一人称视角
  - 至少一段对话或场景描写
  - 中段一句话点穿规律
  - 结尾回到个人决策，不拔高到价值观

## 四、修仙/武侠/历史调料库

刘备教授会在严肃话题里嵌入一两个修仙/武侠/历史比喻，作为"调料"——但 1-2 处就够，超量会喧宾夺主。

### 常用比喻清单
- **修仙类**：灵根、宗门、降维打击、釜底抽薪、修罗场、众神时代、御剑飞行
- **武侠类**：左冷禅、华山派、葫芦娃救爷爷、菜鸟互啄、风卷残云、影武者（黑泽明电影）
- **历史/文学类**：织田信长本能寺、《让子弹飞》、《繁花》、《置身事内》、《老人战争》、《抓娃娃》、《浪浪山小妖怪》、奥本海默众神时代
- **战争/军事类**：缝合怪、内鬼、画 K 线、夺岛、空中撕咬、出师未捷、众神时代

### 调料用法示例（节选自真实样本）
- "感觉就是港村现在的钱太多了，足够抱团几个龙头股起来"（修仙抱团术）
- "韭菜们如果实在难受，可以翻到昨晚夜报再吃一遍鸡汤"（鸡汤=修仙真气）
- "本届大哥是我见过水平最高的"（武侠门派"本届")
- "把美元当成了大哥"（黑社会比喻）
- "葫芦娃救爷爷，直接成建制被俘虏"（神话调料）

## 五、个人锚点系统

作者反复使用几个固定的个人锚点（让读者形成"老朋友"认知）：

| 锚点类型 | 作者原文 | 是否可抄 |
|---|---|---|
| 入市时间 | 2014 年 10 月，筹 50 个起步 | **不要直接抄** |
| 杭州房价 | 14 年单价 4 万 → 巅峰 8 万 → 现在 4 万多 | 仅供节奏参考 |
| 大学住宿 | 大一八人间失眠、大二上铺、大三研究生四人间 | 不要直接抄 |
| 账户结构 | 港股大账户分散、A 股账户、几个小账户单吊 | 仅供节奏参考 |
| 月初仪式 | 每月 1 日收稿费、年底打赏冲数字 | 不要直接抄 |
| 童年朋友 | "和学神做哥们" | 不要直接抄 |

**模仿建议**：替换成自己的可重复锚点（自己的学历、家乡、工作经历、爱好），不要抄作者的具体细节，否则读者一眼看穿是抄袭。
