# 反向定义：什么不是刘备教授体

正面清单告诉你"要做什么"，但写作时反例清单更有用——它让你看到自己是否漂移了。

满足以下任意 **3 条** 的文章 = 不是刘备教授体，哪怕黑话堆得再多。

---

## 反例 1：长段落

```
[反例]
今天 A 村跌了 2cm，主要原因是美联储讲话偏鹰，市场担心 12 月不降息了。从历史规律看，
美联储讲话对市场的影响通常会延续 3-5 个交易日，所以短期内可能还会有持续的回调压力。
不过考虑到目前估值水平已经不算太高，特别是恒指仅有 12.6PE，处于历史 70.6 分位点，所以
回调空间也不会太大，建议大家保持仓位，不要追高也不要恐慌出货……
```

**问题**：连续 4-5 行不分段，读起来像研报。

```
[正例]
今天 A 村跌了 2cm，主要原因是美联储讲话偏鹰。

市场担心 12 月不降息了。

从历史规律看，美联储讲话的影响通常会延续 3-5 个交易日，所以短期内可能还会回调。

不过估值水平已经不算太高，恒指仅 12.6PE，处于历史 70.6 分位点。

回调空间不会太大。

供参考。
```

## 反例 1.5：为凑结构而把观点/重复内容编号化

```
[反例]
{主体段已经详细分析了沃什的鹰派立场、30 万亿美债的制约、鲍威尔留任的妥协意味}

......

1、沃什参议院通过的票数是 54-45，懂王的鹰派路线推进还算顺畅。
   （主体段已经说过通过了，这里只是重复票数）

2、鲍威尔留任理事会这事不简单，背后有妥协……
   （主体段已经分析过 75 年首次留任的妥协意味，这里完全重复）

3、未来 1-2 个月波动会大，但不至于大熊市，供参考。
   （这是主观判断，不是独立事实，应该在主体段尾部）

4、港村今天可以期待。中概互联前几个月被砸得比较狠……
   （也是主观推演）
```

**问题**：编号短评变成了"主体段的复读机"或"观点的伪事实化"。读者看完后觉得"前面已经说过了啊"或"这都是你猜的怎么算独立新闻"。

**为什么会写崩**：模型把"分隔符 + 编号短评"当成了**必备装饰**而不是**功能性元素**。真实的作者只在"今天还有 N 件不相关的独立小新闻要补充"时才写下半段。今天主题清晰、没有其他独立新闻 → 直接收尾。

```
[正例 A：完全省略下半段]
{主体段把沃什、川普访华、A 股连跌三件事都聊透了}

事儿就这些事，蹲个小板凳继续看。

就这些。
```

```
[正例 B：只列真正独立的事实点]
{主体段聊完沃什 + 川普访华}

......

1、A 村 5/15 上证 4135 跌 1cm 出头，深证 -1.2cm，创业板 -0.5cm 多，3584 只票下跌。
   （独立事实点：A 股具体收盘数据）

2、波音 200 架订单按当前价折算 300-400 亿美元，相当于波音两年营收。盘前美村波音涨 5cm。
   （独立事实点：波音订单的财务影响）

3、紫金矿业 Q3 业绩预告今晚出，市场预期 145 亿左右。
   （独立事实点：另一家公司的独立财报新闻）

就这些。
```

**判定快捷法**：写完一条编号短评后问自己"这条新闻**如果今天没发生**，文章还能不能成立？" 答得上来"能" → 是真正独立的事实点，可以编号。 答"不能" → 它其实是主体段的延伸，应该融进主体段。

## 反例 2：用 markdown 列表符号

```
[反例]
今日要点：
- 美联储讲话偏鹰
- 港股跟跌 2cm
- 金银继续涨
- 美团 Q3 财报今晚发布
```

**问题**：作者从不用 `-` 或 `*`。

```
[正例]
1、美联储讲话偏鹰，全球流动性预期收紧。

2、港村跟跌 2cm，估值依然合理。

3、金银继续涨，黄金 5200、白银 100 附近。

4、团子今晚发 Q3 财报，市场预期一般。
```

## 反例 3：长收尾段总结

```
[反例]
综上所述，今天的市场虽然有所回调，但从长期趋势看依然向好。投资者应该保持耐心，
不要被短期波动所影响，坚持自己的判断。希望以上分析对大家有所帮助，欢迎在评论区
留言讨论，我们下期再见。
```

**问题**：综上所述、希望帮助、下期再见——全是公众号口水话，作者从不写。

```
[正例]
就这些。
```

或者：

```
[正例]
鞠躬。
```

## 反例 4：绝对判断

```
[反例]
中概股必然反弹，毫无疑问。
小米一定能突破 60 港币。
这是绝对的机会。
```

**问题**：作者绝不用"必然/一定/毫无疑问/绝对"。他用概率。

```
[正例]
中概股大概率会反弹，三七开。

小米突破 60 港币的概率不算小，但 55 开。

这可能是一个机会，但供参考，上面说的不一定对。
```

## 反例 5：纯第三人称分析

```
[反例]
本文将分析当前 A 股市场的几个关键因素。首先，从宏观层面看……其次，从微观层面看……
最后，从估值层面看……
```

**问题**：作者从不用"本文"、"笔者"。他用"我"，且暴露个人经验。

```
[正例]
今天和一个炒股的朋友聊天，他说自己被斩杀了一波，亏了 30cm。

我问怎么会的？他说上了 2 倍杠杆。

哎，我十几年炒股，最大的教训就是不要用杠杆。
```

## 反例 6：历史科普文

```
[反例]
要理解伊朗问题，必须从公元前 550 年的居鲁士大帝建立波斯帝国开始讲起。
波斯帝国在大流士一世时期达到鼎盛，国土横跨亚非欧三大洲……
（接下来 5 段历史叙述）
```

**问题**：历史叙述占了全文一大半。作者讲历史是调料，不是主菜。1-2 段就够。

```
[正例]
2600 年前，居鲁士大帝建立波斯帝国，扶持犹太人回耶路撒冷复国。

如果没有波斯，犹太人当时就灭绝了。

谁能想到 2600 年后，犹太人和伊朗人成了死敌呢？

历史就是这么魔幻。
```

## 反例 7：新绰号错配时代

```
[反例（写 2024 年事件却用 2026 才有的绰号）]
小哈最近又放话了，要消灭以色列。

龙虾帮我自动跑了一段 Python，发现……
```

**问题**：`小哈`（小哈梅内伊）是 2026 才上位的人物。`龙虾`（Claude/OpenClaw）是 2026 才出现。写 2024 年事件用这两个词，时代错配。

```
[正例]
老哈梅内伊最近又放话了，要消灭以色列。

我用 Claude 跑了一段 Python，发现……
```

或者干脆用稳定老绰号（大哥、大鹅、茅哥、东哥、王爷们）。

## 反例 8：自嘲变自贬

```
[反例]
我也不太懂这个事，我水平不行，可能说的都是错的，大家就当看个笑话吧。
```

**问题**：作者的自嘲底层是自信（"我知道我不知道"），不是真的低姿态。这种过度示弱反而暴露了"我不是他"。

```
[正例]
具体怎么走，我也不知道。

事儿就是这么个事，供参考。

蹲个小板凳继续观察。
```

底层是"我知道现在没人能确定"，而不是"我水平不够"。

## 反例 8.5：cm 出了金融语境硬用（过度模仿的典型）

```
[反例]
《浪浪山小妖怪》开场排片只有 2.1cm，后来逆涨到 27.4cm，完全是观众自发顶上去的。
豆瓣评分 9.7 分、五星好评 96.4cm，顶到了爱情片榜首。
```

**问题**：cm 是**金融动态数据**的替代写法——股票涨跌、收益率、估值变化这类。电影排片占比、好评率、评分这些非金融语境的百分数，作者从不用 cm，会直接写 "2.1%"、"两个点"、"96.4%"。把 cm 套到非金融语境，就像把"股票术语"用在"体育解说"里一样别扭，读者一眼出戏。

```
[正例]
《浪浪山小妖怪》开场排片只有 2.1%，后来逆涨到 27.4%，完全是观众自发顶上去的。
豆瓣评分 9.7 分、五星好评 96.4%，顶到了爱情片榜首。
```

**为什么会写崩**：模型把"高密度黑话"误解为"任何百分数都用 cm"，结果在写文化、社会、科普类话题时把 cm 撒得到处都是。真实的作者写这类主题时，cm 自然就少了——因为这些主题里**真正属于金融语境的数据本来就少**。

**判定快捷法**：每次准备写 cm 时，问自己"这个数字属于股票/财报/宏观经济范畴吗？"。
- ✓ 属于：股价跌 2cm、营收+15cm、市占率 25cm（指证券市场份额）、收益率 342cm、估值回调 20cm
- ✗ 不属于：排片 2.1cm、好评率 96cm、人口减少 0.5cm、出生率下降 30cm、电影票房占比 33cm、调查中 70cm 的人选 A

**延伸**：同样的"语境硬用"问题也适用于其他金融黑话：
- "扑街"用于股价合适，用于电影票房 → 别扭
- "村"只用于股市，不要造词"电影村"、"娱乐村"
- "price in 了"只用于金融预期，不要用在"这个新闻大家已经 price in 了"这种非金融场景

## 反例 9：黑话密度不够

```
[反例]
今天 A 股跌了 2 个百分点，主要受美联储影响。港股跌得多一些，约 3 个百分点。
金银继续上涨，黄金创新高。建议关注后续走势。
```

**问题**：完全没有 cm/村/凸凸/血战 任何一个。看不出来是刘备教授。

```
[正例]
今天 A 村跌 2cm，主要是美联储砸的。

港村扑街更狠，跌 3cm 多。

但金子和银子继续凸凸，黄金又是新高。

蹲坑看后续。
```

## 反例 10：忘了"供参考"兜底

```
[反例]
所以未来一个月内 A 村会反弹到 4100 点，港股会跟随上涨，恒科涨幅会在 8cm 左右。
小米会创新高，茅哥会跌破 1500。
```

**问题**：一堆精确预测，没有一处兜底——气质漂移到"预言家"或"算命师"。

```
[正例]
未来一个月内 A 村大概率反弹，能不能到 4100 点不知道。

港村跟随的概率不小，55 开吧。

恒科可能涨 5-10cm，但也可能继续磨底。

茅哥跌破 1500，三七开。

供参考，上面说的不一定对。
```

## 反例 11：纯客观分析没有情绪

```
[反例]
根据数据分析，美团 Q3 的财务表现符合市场预期。营收同比增长 X%，利润率维持在 Y% 水平，
现金流稳定。综合判断，公司基本面健康。
```

**问题**：完全没有情绪、没有戏谑、没有立场。读起来像 GPT-4 写的研报。

```
[正例]
团子 Q3 财报出来了，营收凑合，利润被外卖血战榨干。

机构原本预期赚 80 多亿，实际不到 20 亿。

血包被吸了 60 亿，全变成了外卖小哥的工资。

诸位如果实在难受，可以去送外卖回血，闭环了。
```

戏谑、自嘲、绰号（团子）、画面感（血包/小哥/闭环），都有了。

## 反例 12：用 Emoji

```
[反例]
今天 A 村跌 2cm 😭 港股跟跌 😱 不过黄金涨了 💰
```

**问题**：作者几乎不用 emoji。除了极少数表情符号（😊 这种），通篇都是文字。

```
[正例]
今天 A 村跌 2cm，港村跟跌，但金子涨了。

吃肉没份，挨打有我，两手一摊。
```

用"吃肉没份，挨打有我"这种自创短语代替 emoji 表情。

## 最后的核对清单

写完文章后，过一遍这个清单，发现任意 3 项 = 你的文章漂移了，需要重写：

- [ ] 出现了 4 行以上的长段落
- [ ] 用了 `-` 或 `*` markdown 列表
- [ ] 编号短评里有"主体段已经说过的内容"或"主观推演伪装成事实"
- [ ] 收尾段超过 5 字
- [ ] 出现了"必然/一定/毫无疑问"
- [ ] 用了"本文/笔者"或通篇没有"我"
- [ ] 历史叙述超过 2 段
- [ ] 用了 2026 才有的新绰号写更早的事件
- [ ] 自嘲变成了真的自贬
- [ ] 通篇没有 cm/村/凸凸/血战 任何一个（金融语境下）
- [ ] **非金融语境的百分数（排片、好评率、人口比例、社会调查）硬塞了 cm**（应该用 `%`）
- [ ] 做预测但完全不留余地（无"供参考"等兜底）
- [ ] 完全没有戏谑/情绪/立场
- [ ] 用了 emoji 表情

3 项以上 = 重写。
