# 素材事实核查指南

> 本文件提供素材核查的具体方法和财经写作中常见的事实错误类型。
> 每条素材入库前都应过一遍本指南中的检查清单。

---

## 核查的基本原则

**宁可不用，不可用错。** 一个错误数据或张冠李戴的引言，会摧毁整篇文章的公信力。读者可以容忍你观点偏颇，但不能容忍你事实出错。

**标注不确定性。** 如果无法100%确认某条素材的准确性，不要假装确定。在素材包中标注 ⚠️ 待核实，并建议用户自行验证后再使用。

---

## 分类核查清单

### 一、数字与数据核查

| 核查项 | 具体操作 |
|-------|---------|
| 数值本身是否正确 | 交叉查证至少两个独立来源 |
| 单位是否正确 | 亿 vs 万亿、美元 vs 人民币、% vs 基点 |
| 时间范围是否准确 | 2023年 vs 2023年Q3 vs 2023年12月 |
| 数据口径是否一致 | 同比 vs 环比、名义 vs 实际、季调 vs 未季调 |
| 数据是否最新 | 确认没有更新版本 |
| 数据出处是否可靠 | 一手数据 > 权威媒体引用 > 二手转述 |

**常见数据错误：**
- 把美元数据当成人民币数据引用（反之亦然）
- 混淆"亿"和"万亿"（中文语境常见）
- 混淆"百分比"和"百分点"（利率从3%降到2%是降了1个百分点，不是降了1%）
- 引用过时数据而未标注时间
- 混淆市值、营收、利润等不同口径的数字

### 二、历史事件核查

| 核查项 | 具体操作 |
|-------|---------|
| 时间线是否准确 | 事件发生的年份、月份、持续时间 |
| 关键人物是否正确 | 名字、职位、所属机构 |
| 因果链是否成立 | 不要把时间先后当因果关系 |
| 结局是否完整 | 不要只讲对自己观点有利的部分 |
| 规模是否准确 | 涨幅、跌幅、受影响人数等 |

**常见历史事实错误：**
- 南海泡沫的年份是1720年，不是1710年也不是1721年
- 郁金香泡沫的严重程度存在学术争议，近年有学者认为被严重夸大
- 1929年大崩盘当日的跌幅常被错误引用（黑色星期四和黑色星期二需要区分）
- 牛顿在南海泡沫中的具体亏损金额有不同版本，需标注来源
- 日本泡沫的最高点是1989年12月29日，日经225指数为38957.44点

### 三、名人引言核查

| 核查项 | 具体操作 |
|-------|---------|
| 是否真的说过 | 用Quote Investigator或原始出处验证 |
| 原文是否准确 | 对照原始文本，不要用转述版本 |
| 语境是否正确 | 不要断章取义 |
| 归属是否正确 | 很多名言被错误归到名人头上 |

**高频被错误引用的名言：**
- "复利是世界第八大奇迹"——常被归为爱因斯坦，但没有可靠证据证明他说过
- "在别人贪婪时恐惧"——巴菲特确实说过，但常被错误引申
- "市场在你能保持偿付能力之前保持非理性"——通常归于凯恩斯，但原始出处存疑
- "我能计算天体运动，却无法计算人类的疯狂"——牛顿是否说过此话存在争议
- "给我一个支点，我就能撬动地球"——阿基米德的话经常被用在不恰当的金融语境中

### 四、理论与概念核查

| 核查项 | 具体操作 |
|-------|---------|
| 理论名称是否准确 | 全称和常用简称 |
| 提出者是否正确 | 核实首次提出的人和时间 |
| 核心含义是否准确 | 参照学术原文，不要想当然 |
| 通俗解释是否失真 | 简化不等于扭曲 |

**常见概念误用：**
- "黑天鹅"不等于"小概率事件"——塔勒布的定义强调的是不可预测性和巨大影响
- "有效市场假说"有弱式、半强式、强式三种形式，不能笼统使用
- "均值回归"不意味着跌了一定会涨——均值本身可能在变
- "沉没成本"在经济学中是理性决策应忽略的成本，但要承认人类确实做不到完全忽略
- "明斯基时刻"特指从稳定转向不稳定的临界点，不是泛指任何市场崩盘

## AI 生成素材的特别注意事项

AI 在搜集素材时有几类系统性风险，搜索结果也可能包含错误信息，需要特别警惕：

### 容易出错的领域
1. **具体数字**：AI 可能编造看似合理但不准确的数字。所有具体数据应通过搜索工具核实。
2. **引言原文**：AI 可能生成"听起来像"某人说的话，但实际并非原文。
3. **时间细节**：AI 可能混淆相近年份的事件。
4. **因果关系**：AI 可能将相关关系表述为因果关系。
5. **程度和规模**：AI 可能夸大或缩小事件的影响程度。

### 应对策略
- 所有具体数字和关键事实，用搜索工具交叉验证后再输出
- 搜索结果如果来自低质量来源（论坛、自媒体），需进一步查证权威来源
- 引言类素材必须标注出处，如果找不到可靠出处，改为转述其观点大意
- 对无法验证的素材统一标注 ⚠️，交给用户最终把关
- 宁可输出"未找到可靠数据"也不编造

## 快速核查流程

对每条素材做如下三步快速检查：

**第一步：来源检查**
→ 这条信息的原始来源是什么？是权威机构还是不明来源？

**第二步：交叉检查**
→ 能否从另一个独立来源找到相同的信息？

**第三步：常识检查**
→ 这条信息是否符合基本逻辑和常识？数量级是否合理？

三步都通过 → ✅ 标记为已核实
任一步有疑问 → ⚠️ 标记为待核实，注明存疑点
