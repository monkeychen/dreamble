# 「聊哉梦呓」素材搜集系统

## 定位与原则

本系统服务于选题确定之后、正文撰写之前的素材准备环节。目标是为每篇文章搭建一个"素材弹药库"，让写作时有料可用、有据可引。

### 三条铁律

1. **事实先行，观点在后。** 所有素材必须经得起查证。数据要有出处，案例要有细节，引言要有原文。宁可少一个素材，不可用一个假素材。
2. **信息增量优先。** 读者随手能搜到的东西不叫素材，叫常识。好素材是读者看完会说"这个我不知道"或"原来是这样"的内容。
3. **服务于观点。** 素材不是堆砌展示博学，每一条素材都要回答一个问题：它帮助论证了文章的什么观点？放不进逻辑链的素材，再精彩也要割舍。

## 素材的六种类型

为「聊哉梦呓」的文章搜集素材时，围绕以下六种类型展开：

### 类型1：历史案例
具体的历史事件，包含时间、地点、人物、起因、经过、结果。这是"以史为镜"内容线的核心燃料。

质量标准：
- 时间和关键数字必须准确（年份、涨跌幅、持续时间）
- 有具体的人物或机构作为故事主角
- 事件逻辑链清晰（因为A → 所以B → 导致C）
- 与当下的映射关系明确

好的历史案例示例：1720年南海泡沫中，牛顿最初投资赚了7000英镑后卖出，但看着股价继续疯涨又追高买入，最终亏损20000英镑。他事后说："我能计算天体的运动，却无法计算人类的疯狂。"

### 类型2：经济/市场数据
具体的数字、统计、指标，用于支撑或反驳某个判断。

质量标准：
- 标明数据来源（国家统计局、央行、Wind、Bloomberg等）
- 标明数据时间范围
- 数据口径清晰（同比/环比、名义/实际、季调/未季调）
- 如有可能，提供趋势变化而非孤立数字

注意事项：
- AI搜索到的具体数字需要交叉验证，标注"待核实"标签
- 优先使用权威机构的一手数据
- 避免使用来源不明的"据统计"

### 类型3：名人观点/经典引言
知名投资者、经济学家、历史人物的原话或核心观点。

质量标准：
- 引言必须有明确的出处（哪本书、哪次演讲、哪封信）
- 优先使用原文，翻译件标注译本
- 引用者本人的身份和背景需简要介绍
- 引言与文章观点的关联要自然，不能硬塞

高频引用人物库：巴菲特、芒格、格雷厄姆、索罗斯、霍华德·马克斯、彼得·林奇、瑞·达利欧、塔勒布、凯恩斯、亚当·斯密、李嘉图、费雪、西格尔、丹尼尔·卡尼曼

### 类型4：学术/理论框架
行为经济学、金融学、心理学等领域的理论概念，用于解释现象背后的逻辑。

质量标准：
- 理论名称准确，提出者明确
- 用通俗语言解释核心含义（读者不是学术同行）
- 附带一个生活化的例子帮助理解
- 说明这个理论对投资决策的具体启示

常用理论框架：锚定效应、损失厌恶、羊群效应、禀赋效应、过度自信偏差、可得性偏差、均值回归、有效市场假说、明斯基时刻、反身性理论、前景理论

### 类型5：类比与隐喻
用其他领域的事物来解释抽象的经济金融概念。这是「聊哉梦呓」文风的重要组成部分。

质量标准：
- 类比的映射关系准确（A的哪个特征对应B的哪个特征）
- 类比来源是读者熟悉的领域（日常生活、自然、体育、医学等）
- 类比要揭示本质，而非仅仅"像"
- 一篇文章中类比不超过2-3个，多了会散

好的类比示例：
- "央行放水就像给感冒病人吃退烧药——体温是降下来了，但病因还在。"
- "分散投资就像诺曼底登陆选多个滩头——不把所有兵力押在一个点上。"
- "股市里的止损像开车系安全带——平时觉得多余，出事了能救命。"

### 类型6：时事/新闻素材
当下发生的具体事件、政策文件、企业动态，用于文章的时效性锚点。

质量标准：
- 来源可靠（官方发布、权威财经媒体）
- 信息完整（5W1H）
- 区分事实与评论（标清楚哪些是事实陈述，哪些是媒体观点）
- 注意时效性，标注信息发布时间

## 素材搜集工作流

### 步骤1：明确搜集方向

从用户给出的选题/提纲中，提取需要素材支撑的关键论点。每个论点对应一个"素材需求"。

操作方法：
- 列出文章的3-5个核心论点
- 为每个论点标注最需要哪种类型的素材（历史案例？数据？名人观点？）
- 判断哪些素材必须有（文章骨架），哪些锦上添花（文章调味料）

### 步骤2：分层搜索

按以下优先级搜索素材：

**第一层：核心素材（必须有）**
- 支撑文章核心观点的关键案例或数据
- 没有这些素材，文章立不住

搜索策略：使用精确关键词搜索，交叉验证多个来源

**第二层：增强素材（应该有）**
- 让论证更丰满的补充案例、对比数据、专家观点
- 没有它们文章能成立，但会单薄

搜索策略：扩展相关领域搜索，寻找反面案例

**第三层：点睛素材（最好有）**
- 让文章出彩的金句、巧妙类比、冷知识、反直觉事实
- 这是文章从"合格"到"优秀"的差距

搜索策略：跨领域联想搜索，翻阅经典著作

### 步骤3：素材验证

对搜集到的每条素材做以下检查：

- **准确性**：数字、日期、人名是否正确？交叉查证至少两个来源
- **完整性**：关键信息有没有遗漏？因果链条是否完整？
- **时效性**：这是最新数据吗？有没有更新的版本？
- **版权性**：大段引用是否有合理使用问题？

对于无法100%确认的素材，统一标注 ⚠️ 待核实，并说明存疑之处。

### 步骤4：素材组织与输出

搜集完成后，按结构化格式整理输出（见下方输出格式）。

## 按内容线的素材搜集重点

不同内容线对素材的需求侧重不同：

### 以史为镜
- **核心需求**：完整的历史事件叙事（时间线、关键人物、数据、结局）
- **辅助需求**：当下事件与历史事件的映射对照表
- **搜索重点**：金融史、经济史、人物传记、历次危机复盘
- **常用搜索词**：XX事件始末、XX年金融危机、历史上的XX、XX泡沫

### 投资认知
- **核心需求**：行为金融学/心理学理论 + 市场中的真实案例
- **辅助需求**：投资大师的原话和具体操作案例
- **搜索重点**：行为经济学研究、投资经典书籍、基金/散户行为数据
- **常用搜索词**：XX效应、散户行为数据、XX偏差 投资、巴菲特/芒格 关于XX

### 经济观察
- **核心需求**：最新的经济数据 + 政策原文/要点
- **辅助需求**：历史数据对比、机构解读摘要
- **搜索重点**：官方数据发布、央行公告、权威财经媒体分析
- **常用搜索词**：最新XX数据、央行XX公告、XX政策解读、XX行业趋势

## 搜索策略指南

### 基本搜索原则
- 先广后窄：先用宽泛关键词了解全貌，再用精确词挖掘细节
- 中英双语：很多经济金融领域的经典案例，英文资料更丰富、更准确
- 多源交叉：同一个事实至少查两个独立来源
- 原始优先：尽量找一手资料（原始论文、官方公告、当事人自述），而非二手转述

### 搜索组合技巧
- **事件 + 数据**："南海泡沫 股价涨幅"
- **人物 + 观点**："Howard Marks memo 2008"
- **概念 + 实证**："loss aversion investing study"
- **对比搜索**："日本失去三十年 vs 中国经济"
- **时间限定**：搜索最新数据时加上年份

### 素材不足时的应急策略
如果核心素材搜索困难，可以：
1. 更换切入角度——可能某个角度更容易找到支撑素材
2. 降维使用——用一个小案例代替宏大叙事
3. 坦率标注——告知用户此处需要补充素材，建议查阅的方向
4. 反向思维——找不到正面案例，找反面案例也能论证

## 额外提示

- 素材搜集环节应主动使用搜索工具获取最新、最准确的信息，而非仅依赖模型内置知识。
- 对于历史事件的具体数据（年份、金额、涨跌幅），务必标注来源并建议用户二次核实。AI 在具体数字上可能出错，这是最需要人工把关的地方。
- 素材包不是越多越好。一篇2000-3000字的文章，核心素材3-5条 + 增强素材2-3条 + 点睛素材1-2条就足够了。素材过多反而会让文章失焦。
- 如果用户已经有了文章提纲，素材搜集应严格按照提纲结构来组织，确保每个段落都有对应的素材支撑。
- 搜集过程中如果发现某条素材非常精彩但与当前选题关联不大，可以单独标注为"素材备忘"，建议用户存入素材库留待未来使用。
