# 猫笔刀 Skill 结构优化评估

评估时间：2026-06-05
评估人：Claude Opus 4.6

---

## 一、8维度结构评估

### 1. 工作流清晰度 — 4.5/5

Agentic Protocol 设计得相当好。Step 1 问题分类（需要事实 / 纯框架 / 混合）给了 AI 明确的决策树，Step 2 按四类问题给出研究清单，Step 3 指向心智模型和表达DNA。AI 拿到这个 Skill 基本知道先做什么。

**扣分点**：Step 2 强制要求 WebSearch，但如果运行环境没有搜索工具（比如纯对话模式），Skill 没有给降级路径——是直接跳到 Step 3 用训练语料回答，还是拒绝回答？这个空白会让 AI 卡住。

### 2. 边界条件 — 3/5

有一条硬边界写得很好："绝对不推荐具体个股，不做短期预测，不替用户做投资决策"。诚实边界章节列了7条局限。

**扣分点**：
- 07-verification 报告已经指出，"遇到猫笔刀没讨论过的话题时，优先用哪些模型推断"没有明确优先级。当前 Skill 有7个心智模型，但没有说"日本股市"这种超出素材范围的话题应该怎么处理——是大胆推断、还是坦诚说"这事我没研究过"、还是用框架做有限分析？
- 缺少对"纯娱乐/情感类问题"的处理策略。如果用户问"猫笔刀你觉得养猫还是养狗好"，Skill 没有任何指引。
- "兵线没到不冲塔"只作为启发式存在，但什么算"敏感话题"没有具体枚举，AI 的判断标准不一致。

### 3. 检查点设计 — 2.5/5

**这是最弱的维度之一。** 整个 Skill 没有任何显式的自检机制。

具体缺失：
- 回答完成后，没有"检查是否违反了角色规则"的 checklist
- 没有"这段话读起来像不像猫笔刀"的风格自检
- 没有"是否不小心推荐了个股 / 做了短期预测"的红线检查
- 没有"数据是否已经过期"的时效性检查
- 表达DNA章节列了很多规则，但没有说"回答后对照以下几项做一次快速扫描"

对比：好的 Skill 会在工作流最后加一个 Step 4: Self-Check，列3-5个必须通过的检查项。猫笔刀 Skill 完全没有这一层。

### 4. 指令具体性 — 4/5

表达DNA章节的具体性很好——句式长度（10-25字）、段落上限（300字）、排比封顶（3-4句）、高频口语词清单、禁忌词清单、确定性表达分三级、三大收束词。这些都是可直接执行的指令。

**扣分点**：
- "短句密集轰炸为绝对主力"——"绝对主力"是多少比例？80%？90%？AI 理解可能不一致
- 幽默章节只给了示例，没给生成规则。"自嘲是核心人设"——什么时候该自嘲、什么话题不适合自嘲，没有说
- "脏话分寸：中等偏低，只在情绪高点出现"——"情绪高点"的判断标准是什么？市场暴跌算吗？用户问无聊问题算吗？

### 5. 心智模型可操作性 — 3.5/5

7个模型本身质量很高，每个都有证据、应用和局限。但触发条件不够明确。

**问题**：
- 模型之间的优先级和组合规则缺失。"黄金暴涨"应该触发成本思维镜？还是非对称博弈？还是两个都用？
- 没有"一个回答通常用几个模型"的指导。用太多会变成模型展览，用太少会显得思考不深
- 启发式和模型的关系不清晰。"好消息先泼冷水"是一个独立的决策规则，还是成本思维镜的子集？

**做得好的地方**：每个模型的"应用"小节给了触发信号（如"遇到任何看似诱人的机会"触发成本思维镜），只是分散在各处，没有汇总成决策表。

### 6. 决策启发式的场景覆盖 — 4/5

9条启发式覆盖了：牛熊转换（#1踏空也是风险）、风险管理（#2输不起的筹码）、交易计划（#3判断+预案）、市场阶段判断（#4能算出来的都是熊市）、利好解读（#5好消息先泼冷水）、信息可靠性（#6兵线没到不冲塔）、跟单陷阱（#7别抄作业）、概率判断（#8链上赌场）、宏观消费（#9房价不稳消费不起）。

**覆盖不足的场景**：
- "该不该止损"——散户最常遇到的具体操作问题，9条里没有直接对应
- "新手入门该怎么学投资"——教育类问题没有启发式
- "现在该买什么"——虽然有"不推荐个股"的铁律，但怎么引导用户转向框架思考，没有启发式

### 7. 角色一致性 — 4.5/5

整体语气统一度很高。身份卡、心智模型、表达DNA、价值观各章节的语调一致——直接、口语化、不装。

**轻微不一致**：
- "核心心智模型"章节的写法偏学术（"证据""应用""局限"的三段式结构），与猫笔刀本人"不当老师"的人设有微妙冲突。不过这是写给 AI 的指令而非输出给用户的内容，所以影响有限。
- "智识谱系"这个词本身就不像猫笔刀会用的词，太文绉绉了。但同样，这是元数据层，不影响输出。

### 8. 信息密度 — 4/5

**冗余**：
- "人物时间线"和"最新动态"有部分重叠（如"2024年下半年BTC只卖不买"在时间线和身份卡中各出现一次）
- "附录：调研来源"和"关键引用"对 AI 执行 Skill 的帮助有限，主要是给人类审查用的，占了约40行

**遗漏**：
- 没有"猫笔刀绝对不会说的话"的负面清单（禁忌词只在表达DNA里列了一些，但没有完整的反面教材）
- 缺少"典型回答模板"——给 AI 一个完整的示范回答，比列一堆规则更有效

## 二、评分汇总

| 维度 | 评分 | 一句话评价 |
|------|------|-----------|
| 1. 工作流清晰度 | 4.5/5 | 决策树清晰，缺无工具时的降级路径 |
| 2. 边界条件 | 3/5 | 硬边界有，软边界（超出范围话题、非财经话题）缺失 |
| 3. 检查点设计 | **2.5/5** | **最弱。完全没有自检机制。** |
| 4. 指令具体性 | 4/5 | 表达DNA很具体，幽默和脏话的触发条件模糊 |
| 5. 心智模型可操作性 | 3.5/5 | 模型质量高，但触发条件和组合规则缺失 |
| 6. 启发式场景覆盖 | 4/5 | 9条覆盖广，止损、新手入门场景缺失 |

**总分：30/40（75%）——结构扎实，两个明显短板需要补强。**

## 三、干跑测试

### Prompt 1: "猫笔刀，你怎么看最近黄金暴涨？"

**问题类型**：混合问题（需要最新数据 + 框架分析）

**Skill 引导路径评估**：
1. Step 1 分类 → 混合问题 → 先获取事实再用框架分析。正确。
2. Step 2 研究 → 属于"看市场/宏观"，但黄金同时涉及"看国际事件"（地缘政治避险）。Skill 没有指导"跨多个研究模板"时怎么合并。AI 可能只走一个模板，漏掉另一个维度。
3. Step 3 回答 → 预期触发模型：成本思维镜（黄金开采成本支撑）、非对称博弈（地缘冲突推高避险需求）、时间补偿思维（黄金定投是时间补偿策略）。但 Skill 没有告诉 AI 该组合这三个模型，AI 可能只用一个。
4. 启发式 → #5"好消息先泼冷水"应当触发（黄金暴涨是好消息）。这条写得够清楚，AI 大概率能命中。
5. 表达DNA → 猫笔刀本人在定投黄金，应该会先说自己的仓位（透明原则），再分析为什么涨，最后泼冷水。Skill 的"身份卡"和"最新动态"提供了持仓信息，AI 有素材可用。

**结论**：引导路径基本正确，但模型组合规则的缺失可能导致分析维度不够丰富。中等风险。

---

### Prompt 2: "我想辞职全职炒股，你觉得靠谱吗？"

**问题类型**：纯框架问题（人生决策）

**Skill 引导路径评估**：
1. Step 1 分类 → 纯框架问题 → 跳到 Step 3 直接用心智模型。正确。但这个问题其实也可以走 Step 2 "看个人决策"的研究清单（成本收益比、时间补偿、信息差、含中量）。Skill 的分类表对"人生决策"该走哪条路径不够明确。
2. 预期触发模型：含中量定律（A股散户池含中量极高）、信息差猎人（你比机构多知道什么？）、行为金融学镜片（全职后情绪压力更大，偏差更容易放大）。
3. 启发式 → #2"不要玩输不起的筹码"（辞职 = 把全部收入来源压在炒股上）、#7"别抄作业"（别看谁谁谁全职炒股赚了就跟）。
4. 价值观 → "不当老师"但"实际一直在教"的内在张力——猫笔刀会给框架分析，但不会直接说"别辞"或"辞吧"。Skill 的"投资建议：给框架而非指令"写得足够清楚。

**结论**：引导路径正确，Skill 提供了足够的素材让 AI 给出高质量回答。低风险。这是 Skill 最擅长的问题类型。

---

### Prompt 3: "帮我分析一下宁德时代这只股票"

**问题类型**：触碰"不推荐个股"红线的问题

**Skill 引导路径评估**：
1. 角色规则明确写了"绝对不推荐具体个股"——这是铁律。AI 应该拒绝直接分析个股。
2. **但拒绝之后怎么办？** Skill 没有给"拒绝后的替代方案"。好的处理是：拒绝推荐个股 → 转向行业框架分析（新能源产业链的含中量、产能过剩周期）→ 给用户自己判断的工具。
3. 猫笔刀本人的处理方式：他确实讨论过行业（光伏、新能源车产能过剩），但避开个股代码。Skill 有"看公司/行业"的研究清单，其中"产能过剩程度""含中量评估"等条目暗示了可以讨论行业层面。
4. **风险**：AI 可能生硬地说"我不推荐个股"就结束了，像一个合规机器人而非猫笔刀。猫笔刀的风格是拒绝但给你更好的东西——"我不分析个股，但新能源这个赛道我可以跟你唠唠……"

**结论**：红线足够清晰，但缺少"优雅降级"的具体指导。AI 可能拒绝得太生硬，不像猫笔刀。中等风险。

## 四、最弱两个维度 + 改进建议

### 最弱维度1：检查点设计（2.5/5）

**问题**：整个 Skill 没有自检机制，AI 输出后无法验证是否跑偏。

**改进方案**：在"回答工作流"末尾增加 Step 4: 自检。

**直接可替换的文本——在 SKILL.md 的 `### Step 3: 猫笔刀式回答` 之后插入：**

```markdown
### Step 4: 输出前自检（必须执行）

回答完成后，在输出前对照以下检查项，任何一项不通过则修改后再输出：

1. **红线检查**：有没有推荐具体个股？有没有做短期价格预测？有没有替用户做决策？→ 有则删掉，改为框架引导
2. **角色检查**：读一遍回答，有没有出现"猫笔刀可能会认为""从猫笔刀的角度来看"这类跳出角色的表达？→ 改为第一人称直接判断
3. **风格检查**：
   - 有没有超过25字的长句？→ 拆短
   - 有没有"综上所述""需要注意的是""总的来说"等AI腔？→ 换成猫笔刀的口语
   - 有没有用收束词结尾（"上钟""舅酱""发射"中选一个）？→ 日报式回答加收束词，简短回答可不加
4. **数据时效检查**：引用了Skill中的持仓数据或市场数据吗？→ 如果引用了，必须标注"截至2026年6月的信息，当前可能已变化"
5. **泼冷水检查**：如果回答中有利好判断，有没有至少一句风险提示或反面观点？→ 没有则补一句
```

---

### 最弱维度2：边界条件（3/5）

**问题**：超出素材范围的话题、非财经话题、工具不可用时，AI 不知道怎么降级。

**改进方案**：在"角色扮演规则"章节末尾增加降级策略。

**直接可替换的文本——在 SKILL.md 的 `**退出角色**：用户说「退出」「切回正常」「不用扮演了」时恢复正常模式` 之后插入：**

```markdown
### 降级策略

**超出素材范围的财经话题**（如日本股市、印度经济等猫笔刀未明确讨论过的领域）：
- 先坦诚："这个方向我没有深入跟踪过，不太好说。"
- 再尝试用最相关的1-2个心智模型做有限推断（优先选含中量定律和信息差猎人，这两个模型的泛化能力最强）
- 最后收束："这纯属我拿框架套的，别当投资建议，你们自己判断。"

**触碰个股红线的问题**（如"分析一下XX股票"）：
- 不要生硬拒绝。先说"我不分析个股，这是铁律"
- 然后主动转向行业层面："但这个行业/赛道我可以跟你聊聊"
- 用"看公司/行业"研究清单中的产能过剩、含中量、成本结构等维度做行业分析
- 最终引导用户自己建框架："你把这几个问题想清楚了，要不要买你自己心里就有数了"

**非财经话题**（如生活、情感、育儿等）：
- 猫笔刀本人确实聊这些（儿子对话、消费观、台州老家），可以接话
- 用猫笔刀的语气和价值观回应，但不要强行套心智模型
- 保持自嘲和直男视角，不鸡汤不说教

**无搜索工具时**：
- 如果问题被分类为"需要事实的问题"或"混合问题"，但运行环境没有 WebSearch 等工具：
- 坦诚告知："这个问题需要最新数据支撑，我手头没有实时信息，只能用框架聊个大概方向。"
- 跳到 Step 3，基于心智模型做定性分析，但明确标注"缺少最新数据，定性判断仅供参考"
- 绝不编造具体数字或伪装成最新信息
```

## 五、改进优先级

| 优先级 | 改进项 | 预期效果 |
|--------|--------|---------|
| P0 | 增加 Step 4 自检机制 | 防止角色跑偏、AI腔泄漏、红线越界 |
| P0 | 增加降级策略 | 覆盖超范围话题、个股红线、非财经话题、无工具环境 |
| P1 | 增加心智模型触发表 | 让 AI 知道什么问题用什么模型、一次用几个 |
| P2 | 精简附录/关键引用 | 减少约40行低效内容 |
| P2 | 增加一个完整的示范回答 | 比规则列表更直观地教 AI "什么是对的" |
