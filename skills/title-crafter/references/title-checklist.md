# 标题发布前核查清单

> 标题确定后、发布前，逐项过一遍这份清单。
> 全部通过才能发布，任何一项未通过都需要修改。

---

## 一、基础检查（必须全部通过）

- [ ] **长度合适**：标题核心信息在前30个字以内（微信手机端显示限制）
- [ ] **没有错别字**：逐字检查，尤其注意同音字混淆
- [ ] **没有歧义**：标题不会被误读为另一个意思
- [ ] **独立成立**：不依赖上下文或配图就能理解标题在说什么
- [ ] **事实准确**：标题中涉及的数字、人名、时间等信息无误

## 二、红线检查（任一项触发即否决）

- [ ] 不包含具体个股名称或买卖暗示
- [ ] 不预测具体点位或涨跌方向
- [ ] 不制造恐慌或贩卖焦虑
- [ ] 不做无法兑现的承诺
- [ ] 不包含平台敏感词（可能触发审核）
- [ ] 不存在标题党嫌疑（正文确实能兑现标题的暗示）

## 三、效果检查（至少通过4项）

- [ ] **好奇心测试**：把标题发给一个不了解你公众号的朋友，他会不会想点进去？
- [ ] **朋友圈测试**：如果这个标题出现在你的朋友圈，它能不能从一堆内容中跳出来？
- [ ] **口语测试**：你能不能自然地对朋友说"我刚看了一篇文章叫[标题]"？如果说出来别扭，标题可能太书面或太做作。
- [ ] **隔天测试**：如果明天才看到这个标题（不是刚写完正文的你），还会想点吗？
- [ ] **搜索测试**：标题中是否包含目标读者可能搜索的关键词？
- [ ] **竞品测试**：和其他财经号今天发的标题放在一起，你的标题有没有辨识度？

## 四、调性检查

- [ ] 这个标题听起来像「聊哉梦呓」会说的话吗？
- [ ] 标题的情绪基调和正文匹配吗？（不要标题很沉重，正文很轻松，反之亦然）
- [ ] 标题展现了你的独特视角吗？（还是换成任何一个财经号都能用？）
- [ ] 标题的语气是平视交流的吗？（不是俯视教育，也不是仰视讨好）

## 五、技术优化

- [ ] **前7个字是否有力**：微信消息通知只显示标题开头几个字，前7个字必须能抓住注意力
- [ ] **标点使用得当**：问号、冒号、破折号等标点用于强化节奏，而非堆砌
- [ ] **避免过度使用符号**：不要用【】、！！、……等符号来制造视觉刺激
- [ ] **排除同质化**：检查最近5篇文章的标题，确保没有在重复同一种标题模式

---

## 快速决策流程

当在多个备选标题之间犹豫不决时，按以下流程做最终选择：

**第一步：淘汰**
先把触碰红线的全部淘汰，再把基础检查不通过的淘汰。

**第二步：排序**
对剩下的标题，回答一个核心问题：
> "如果我是一个刚打开微信、在订阅号列表里快速滑动的散户，哪个标题会让我的手指停下来？"

**第三步：确认**
选定后做最后一次"承诺检查"：这个标题暗示的内容，我的文章确实能给到吗？如果能，发。如果有一丝犹豫，要么改标题，要么补正文。

---

## 标题失败后的复盘模板

如果一篇文章的打开率明显低于预期，对标题做以下复盘：

1. **钩子是否足够**：标题有没有给读者一个"非点不可"的理由？
2. **信息是否精准**：读者能不能从标题准确预判文章内容？如果预判不到，他可能觉得跟自己无关。
3. **是否过于通用**：这个标题是不是换个号也能用？没有辨识度的标题打开率通常不高。
4. **竞品干扰**：同一天是否有大量同类话题的文章？标题是否在竞争中被淹没了？
5. **发布时间**：标题是否匹配发布时间的读者心理状态？（比如早上通勤的读者和深夜刷手机的读者，心理状态不同）

复盘结论记录下来，积累成标题优化的经验库。
