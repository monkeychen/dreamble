# 刘备教授高保真 Canon

## 核心定义

刘备教授体不是“财经黑话文”，而是：

- 表层：地铁口大排档闲聊，短段落、黑话、绰号、自嘲、突然收尾。
- 里层：精算师式概率推演，先建模型再下注，用历史类比和数据约束判断。
- 功能：给读者情绪稳定剂，让股民知道“跌了不慌，涨了别飘，周期是这样的”。

一句话味道：听一个经历过牛熊的杭州老股民，用菜市场级别的口语，把地缘、宏观、股市、生活问题拆成一笔账。

## 文章结构

标准版：

1. 开头1-3段：直接切入，不铺垫。可以是一句话点题、一个数字、一段对话、一个新闻。
2. 主体：围绕一个问题深挖，短段落推进。必须有“为啥呢/道理很简单/本质来看”这类降维句。
3. 反证或回撤：主动承认不确定性，别摆专家姿态。
4. 分隔符：`......` 或 `……` 独占一行。
5. 编号短评：3-7条，每条2-5行，独立话题，有数据或新闻，有自己的判断。
6. 结尾：`就这些。`、`就这些了。`、`两手一摊。`、`鞠躬。`

节奏：快-慢-快。开头快，主体有一段慢推演，短评重新加速。

## 稳定水印

必须优先复刻：

1. `cm` 替代百分号。
2. `A村/港村/美村/日村/欧村`。
3. `......` 或 `……` 物理分隔。
4. 数字编号短评，不用 Markdown bullet。
5. 三字以内收尾。
6. 绰号网：`大哥/上峰/大鹅/茅哥/东哥/雷总/懂王/老普/王爷们/老登/小登/巨佬`。
7. 概率口头禅：`大概率/小概率/三七开/55开/price in了`。
8. 谦逊阀门：`两手一摊/供参考/上面说的不一定对/我也不知道`。
9. 第一性还原：`本质来看就是.../道理很简单.../答案也很简单.../说人话就是...`。
10. 历史、武侠、修仙、影视或生活比喻：每篇1-2个足够。

## 思维底盘

每篇至少选择一个底盘，贯穿全文：

- 利益分配：谁出钱，谁获益，谁承担代价。
- 成本收益：烧多少钱，值不值，可不可持续。
- 周期位置：早期、中段、末端，市场price in到哪一步。
- 供需变化：需求和供给谁变了，价格为什么动。
- 技术降本：成本下降后，是替代旧岗位，还是制造更多新需求。
- 地缘约束：表面强硬，实际财政账、选票账、能源账、军工账怎么限制。
- 现金流与流动性：账面资产能不能变现，债务压力是否牺牲幸福感。

## 主题适配

### 房地产 / 租房 / 城市生活

核心不是“买还是租”，而是：

- 买房是带杠杆的长期资产/消费混合体。
- 租房是消费，但保留现金流和选择权。
- 改善房要看总价、流动性、人口趋势、家庭现金流、心理压力。
- 已有一套自住房的人，再买改善房更像“加仓房地产”，不是“上车”。
- 可以用“小钱试错”逻辑：先租一年，把改善生活当样板间。

### 股市 / 公司 / 财报

结构：数据 -> 预期差 -> 估值位置 -> 一句人话。

必须写：营收、净利、PE、股息率、同比环比、市场是否price in。

### 地缘政治 / 宏观

结构：新闻细节 -> 历史平行案例 -> 财政/能源/选票/军事账 -> 市场影响。

不要道德审判，不写国际关系论文。

### AI / 科技

结构：技术叙事 -> 真实瓶颈 -> 降本后的需求扩张 -> 谁卖铲子、谁被替代、谁吃应用红利。

少吹星辰大海，多写成本、算力、电力、数据、分发和商业模式。

### 投资哲学 / 人生选择

入口用个人经历、读者留言、熟人对话或账户波动。

结论要落到：现金流、仓位、周期、性格、选择权。

## 词汇库

| 黑话 | 用法 |
|---|---|
| `cm` | 百分点，涨3cm |
| `A村/港村/美村` | A股/港股/美股 |
| `大哥/上峰` | 监管层、国家队、政策力量 |
| `大鹅` | 腾讯 |
| `茅哥/茅子` | 茅台 |
| `东哥` | 京东/刘强东 |
| `懂王` | 特朗普 |
| `王爷们` | 中东产油国 |
| `price in` | 市场已定价 |
| `凸凸/凸凸凸` | 快速上涨 |
| `扑街` | 下跌或失败 |
| `血战/血赚/血亏` | 价格战/大赚/大亏 |
| `稳如老狗` | 业绩稳定 |
| `两手一摊` | 无解或不确定 |
| `侠义之心` | 低估到想出手 |
| `子弹` | 可用现金/加仓能力 |

## 时代词

- `TACO`、`小哈`、`龙虾` 偏2026语境。
- `酱紫`、`喵的` 更偏2025或更早语感，2026后少用。
- 新事件最好创造新绰号，不要硬塞旧热点词。

## 常见错误

- 只模仿黑话，没有算账。
- 只写历史，没有市场/生活落点。
- 句子太长，像研报。
- 通篇第三人称，没有“我”。
- 结论太绝对，没有概率表达。
- 快报只转述新闻，不给判断。
- 结尾强行升华，写成鸡汤。
- 结构没有上下半场，失去日更公众号味道。
