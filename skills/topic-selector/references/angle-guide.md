# 热点选题角度切入指南

> 本文件解决一个核心问题：当一个热点事件出现，所有财经号都在写，「聊哉梦呓」如何找到自己的差异化角度？

---

## 角度切入的核心原则

**不追信息，追认知。**

热点刚出的前2小时，拼的是速度，那是新闻媒体和快讯号的战场。「聊哉梦呓」不跟这个节奏。我们等信息沉淀后，提供的是：

1. 别人没看到的角度
2. 放进更大框架后的理解
3. 读完之后"哦原来是这样"的认知增量

---

## 六种差异化角度模板

### 角度1：历史平行法
找到历史上的类似事件，做对照分析。

操作步骤：
- 这个事件的核心变量是什么（政策？资金？情绪？技术？）
- 历史上有没有相同核心变量驱动的事件
- 那次的过程和结果是什么
- 今天与那次的相似点和不同点分别是什么
- 如果历史"押韵"，接下来可能怎么演

示例：央行突然降息 → 回溯过去5次突发降息后的市场走势、经济背景、持续时间

### 角度2：利益链拆解法
任何经济事件背后都有利益的重新分配。拆清楚谁受益、谁受损、谁在博弈。

操作步骤：
- 列出这个事件涉及的所有利益方
- 每一方的诉求是什么
- 谁是赢家、谁是输家
- 博弈均衡点可能在哪里
- 对投资者来说，应该站在哪一方的视角

示例：某行业监管新政 → 政策制定者要什么、行业龙头要什么、中小企业要什么、消费者要什么、投资者应该看什么

### 角度3：第二层思维法
当市场有一个主流叙事时，思考这个叙事的反面或下一步。

操作步骤：
- 当前市场的主流观点/叙事是什么
- 这个叙事被price in了多少
- 如果主流观点是对的，接下来会发生什么（已经反映在价格里了吗？）
- 如果主流观点是错的，风险在哪里
- 有没有大家忽略的变量

示例：市场普遍认为"降息利好股市" → 但历史上降息初期往往对应经济下行确认，市场可能先跌后涨

### 角度4：人性/行为金融学透镜
用行为经济学和心理学的框架来解释市场参与者的行为。

操作步骤：
- 当前市场参与者的主流情绪是什么
- 这种情绪对应哪种认知偏差（锚定、从众、损失厌恶、过度自信等）
- 这种集体偏差通常会导致什么后果
- 理性投资者应该如何应对

示例：某只股票暴跌后散户恐慌割肉 → 从损失厌恶和恐慌传染的角度解释，引出逆向投资的思考

### 角度5：数据反直觉法
用数据推翻一个"常识性"判断。

操作步骤：
- 关于这个事件，大家的"常识性"认知是什么
- 找到与常识相反的数据或案例
- 解释为什么直觉是错的
- 给出基于数据的正确理解

示例："降准一定利好银行股" → 拉出历次降准后银行股表现数据，发现并非如此

### 角度6：跨领域类比法
用一个完全不同领域的框架来理解这个事件。

操作步骤：
- 这个经济/金融事件的本质结构是什么
- 有没有其他领域（生物学、物理学、军事、体育、日常生活）有类似结构
- 用那个领域的语言重新描述这个事件
- 这个类比能给读者什么新的理解

示例：A股的"政策底—市场底—经济底" → 类比为医学中的"用药—退烧—康复"三个阶段

## 热点响应的时间策略

| 时间窗口 | 适合做什么 | 不适合做什么 |
|---------|-----------|------------|
| 事件发生当天 | 快速判断是否值得写、确定角度、搜集素材 | 匆忙发文（除非角度已经非常成熟） |
| 第2天 | 发布深度分析，此时信息已沉淀但热度仍在 | 重复已有的分析 |
| 第3-5天 | 发布"冷思考"型文章，反思主流叙事 | 炒冷饭 |
| 一周后 | 做复盘总结，提炼经验教训 | 假装事件还是新闻 |
| 一个月后 | 回头看当时的预判是否正确，做认知校准 | - |

---

## 判断一个热点值不值得跟的清单

逐项检查，满足3个以上才值得跟：

- [ ] 这个事件影响面广，不只是圈内话题
- [ ] 我的读者会因为这个事件产生困惑或焦虑
- [ ] 我能提供不同于快讯和研报的视角
- [ ] 这个话题有足够的纵深可以展开（不是一句话就说完了）
- [ ] 事件背后涉及可复用的投资逻辑或认知框架
- [ ] 有合适的历史对照或数据支撑
- [ ] 发出来后内容保质期至少一周以上

如果只满足1-2个，可以在其他文章中顺带提及，但不值得单独成篇。
