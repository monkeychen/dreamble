# 财经写作高频事实陷阱

> 本文件列出了财经类公众号文章中最常出现的事实错误类型。
> 审校时按本清单逐类扫描，可以捕获80%以上的常见错误。

---

## 一、数字类陷阱

### 1. 中英文数量单位混淆

这是财经写作中最隐蔽的错误之一。

| 错误 | 原因 | 正确做法 |
|------|------|---------|
| "公司市值100亿美元"实际是100 billion美元（1000亿美元） | 英文billion=十亿，不是"亿" | 英文数据必须手动换算：1 billion = 10亿 |
| "GDP增长3万亿"没标币种 | 混用人民币和美元 | 所有金额必须标注币种 |
| "亏损2万英镑"实际是"20000英镑" | 中文"万"和英文计数习惯不同 | 核查原始来源的计量单位 |

### 2. 百分比 vs 百分点

这个错误出现频率极高，甚至专业财经媒体也会犯。

- ❌ "利率从3%降到2%，降了1%" → 实际降了1个**百分点**，降幅是33%
- ❌ "通胀率上升了2%" → 是上升了2个百分点还是在原有基础上上升了2%？
- ✅ 表述要求：涉及利率、通胀率等百分比数据的变动，必须区分"百分点"和"百分比"

### 3. 涨跌幅的计算基准

- ❌ "从100涨到200再跌到100，涨了100%又跌了100%"→ 实际跌幅是50%
- ❌ "某股票先跌50%再涨50%就回本了" → 实际只回到了75%
- ✅ 涨跌幅必须明确基准：从什么价格、到什么价格、涨跌幅多少

### 4. 名义值 vs 实际值

- "1972年的7000英镑"如果不做通胀调整，直接对比今天的购买力是严重误导
- GDP增速要区分名义增速和实际增速（扣除通胀）
- 历史数据如果要跨时期对比，最好标注是否经过通胀调整

### 5. 同比 vs 环比

- 同比：跟去年同期比（消除季节性）
- 环比：跟上一个周期比（反映短期变化）
- 两者数据可能差异很大，不标注口径会误导

## 二、历史事件类陷阱

### 1. 年份偏移

AI最常犯的错误。以下是高频被搞混的年份：

| 常见错误 | 正确信息 |
|---------|---------|
| 南海泡沫在1710年代 | 1720年，股价在1720年夏天见顶 |
| 1929年大崩盘是"黑色星期一" | "黑色星期四"是10月24日，"黑色星期二"是10月29日 |
| 日本泡沫在1990年破裂 | 日经指数在1989年12月29日见顶（38957点），1990年开始下跌 |
| 2008年金融危机始于雷曼倒闭 | 雷曼是2008年9月倒闭，但危机信号在2007年已出现（次贷问题） |
| A股6124点是2008年的 | 2007年10月16日，6124.04点 |
| A股5178点是2015年6月12日 | 正确，这个日期经常被搞混为6月15日（那是暴跌开始的日子） |

### 2. 涨跌幅度的多个版本

同一事件在不同来源中数据可能不同，取决于：
- 计算的起止日期不同
- 用的是收盘价还是盘中极值
- 用的是哪个指数（道琼斯、标普500、纳斯达克）

审校原则：选择最权威来源的数据，并标注计算区间。

### 3. 因果关系的简化

- ❌ "石油危机导致了1973年股市崩盘" → 石油危机是重要因素之一，但不是唯一原因
- ❌ "美联储加息导致2022年A股下跌" → 传导链条远比这复杂
- ✅ 使用"催化""加剧""在XX背景下"等措辞，避免过于简单的因果归因

## 三、名人引言类陷阱

### 1. 高频被错误归属的名言

| 常见误引 | 真实情况 |
|---------|---------|
| 爱因斯坦说"复利是世界第八大奇迹" | 无可靠证据证明爱因斯坦说过此话 |
| 凯恩斯说"市场保持非理性的时间可以比你保持偿付能力的时间更长" | 出处存疑，可能并非凯恩斯原话 |
| 牛顿说"我能计算天体运行，却无法计算人类的疯狂" | 广泛流传但原始出处不确切 |
| 马克·吐温说"历史不会重复，但会押韵" | 无法确认马克·吐温说过此话 |
| 巴菲特说"潮水退去才知道谁在裸泳" | 巴菲特确实说过类似的话，出自2001年致股东信 |
| 丘吉尔说"不要浪费一场好的危机" | 原话出处存疑，可能是后人附会 |

### 2. 引言验证的优先级

- 如果能找到原文出处（哪本书、哪次演讲、哪封信），直接标注
- 如果找不到原文但被广泛转引，使用"据传""常被引述为"等限定语
- 如果完全无法验证，改为转述其观点大意，不用引号

## 四、公司/指数类陷阱

### 1. 同名混淆

- "中国平安"（保险）vs"平安银行"（银行）
- "茅台"可以指贵州茅台（股票）或茅台酒（产品）
- "宝丽来"（Polaroid）——注意有多个时期的公司实体，原公司2001年破产后品牌被转卖

### 2. 指数混淆

- 上证指数 vs 上证50 vs 沪深300 vs 创业板指——引用数据时必须明确是哪个指数
- 道琼斯 vs 标普500 vs 纳斯达克——不同指数同期表现可能差异很大
- "茅指数"不是官方指数，是万得编制的概念指数

### 3. 数据时效

- 市值、PE、股价等数据变化频繁，引用时必须标注日期
- "当前市值XX亿"——这个"当前"是什么时候？

## 五、逻辑类陷阱

### 1. 相关性 ≠ 因果性

- ❌ "历次降准后市场都上涨了，所以降准利好股市"
  → 需要看样本量、时间窗口、其他变量是否被控制
- ✅ "在过去X次降准中，Y次市场在随后一个月内上涨，但也有Z次出现了下跌"

### 2. 幸存者偏差

- ❌ "巴菲特、索罗斯都是长期投资的典范，说明长期投资一定赚钱"
  → 你只看到了成功者，没看到用同样策略失败的人
- ✅ 提及成功案例时，至少简要承认"并非所有采用类似策略的人都取得了同样的结果"

### 3. 基数效应

- ❌ "某行业去年增长50%，今年增长10%，说明行业在衰退"
  → 可能是因为去年基数低（疫情影响等），今年基数正常化
- ✅ 讨论增长率变化时必须考虑基数效应

### 4. 以偏概全

- ❌ "A股散户的平均持股时间只有17天"——这个数据有多个版本，核查具体来源和计算方法
- ✅ 使用汇总性统计数据时标注来源、时间范围、计算口径

## 六、审校快速扫描法

拿到一篇文章后，按以下顺序快速扫描：

**第一遍：扫数字**
用眼睛快速扫全文，标记所有具体数字（年份、金额、百分比、倍数、天数）。这些是最需要核查的。

**第二遍：扫人名和引言**
标记所有提到的人名和直接引语。人名要确认身份正确，引语要确认出处。

**第三遍：扫因果**
标记所有因果表述（"因为""导致""所以""于是"）。每一个因果链都要问：这个因果关系成立吗？有没有遗漏重要变量？

**第四遍：扫程度**
标记所有程度性表述（"所有""从来没有""每次都""一定""永远"）。这些绝对化表述通常需要软化。
